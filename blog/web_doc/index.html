Web-Based User Guide

This web document is based off a printed user guide for a 3D visualization system for ophthalmic surgery. I converted it to a web-based format to demonstrate potential topic organization and modularization in a responsive output format. Parts of this document are fictitious, removed, or redacted for proprietary purposes.

My responsibilities for the original document included interpreting explanations from a Research and Development department acquired by the company, illustrating isometric and 2D drawings of individual system components and combined assemblies from CAD drawings, designing the visual styles based off the corporate brand guidelines and technical communication techniques, and creating the entire document. For the web-based document, I parsed the information into modular topics and converted them to markdown files. I utilized a Hugo theme to create the layout and published it to github.

Programs used:

  • Atom
  • Adobe Illustrator
  • Hugo
